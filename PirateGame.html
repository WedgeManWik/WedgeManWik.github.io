Pirate Game - Personal Project

Unreal Engine


This game is a personal project I've been working on ever since I finshed my first year of university. The inspiration came from League of Legends (a video game), and One Piece (an anime). I thought that I'd finally give programming in a game engine a proper go, and since Unreal Engine 5 just came out at the time, and I wanted to have a bit of a head start to the next academic year, this was a perfect opportunity to get started.

At the moment, the game is nowhere near completion; however, I'm constantly planning and adding features to the game, so hopefully by the time I finish university, the game will be in a state worthy of publishing.
